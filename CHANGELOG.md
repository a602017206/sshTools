## 使用方法

### 新建连接
1. 点击"+ 新建连接"按钮
2. 填写连接信息（名称、主机、端口、用户名）
3. 输入密码（可选，用于测试连接）
4. 点击"测试连接"验证配置（需要密码）
5. 点击"保存"保存配置

### 编辑连接
1. 在连接列表中找到要编辑的连接
2. 点击"编辑"按钮
3. 修改需要更改的字段
4. 点击"保存"更新配置

### 连接到SSH服务器
1. 在连接列表中找到目标连接
2. 点击"连接"按钮
3. 在弹出的对话框中输入密码
4. 等待连接建立

### 删除连接
1. 在连接列表中找到要删除的连接
2. 点击"删除"按钮
3. 确认删除操作
